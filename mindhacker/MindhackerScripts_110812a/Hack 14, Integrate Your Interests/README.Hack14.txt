The script in this folder is the Interest Integrator from Mindhacker Hack 14, "Integrate Your Interests." The simplest way to run this program on a system with a Perl interpreter is to type 

  perl ii

within the directory where the ii script is.

The obsessions.txt file is meant only as an example. We hope you'll write and enjoy your own.
